# 🏥 Sistema de Ficha Médica - H&P / SLB

Aplicación web para el registro de información médica previa al ingreso a locaciones operativas, cumpliendo con requisitos de seguridad industrial y salud ocupacional.

## 🚀 Características

- 📋 Formulario médico completo
- 🧾 Validación de datos obligatorios
- 🌎 Manejo de fechas en horario Colombia
- 🏢 Selección de locación con opción personalizada
- ☁️ Almacenamiento en Supabase
- 🔐 Acceso administrativo (login)
- 📊 Dashboard con exportación a Excel

## 🛠️ Tecnologías utilizadas

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Base de datos: Supabase
- Librerías:
  - requests
  - openpyxl

## 📊 Funcionalidades
- Registro de:
    - Datos personales
    - Antecedentes médicos
    - Enfermedades actuales
    - Contacto de emergencia
- Exportación a Excel:
    - Formato profesional
    - Columnas ordenadas
    - Fechas formateadas
    - ID automático

## 📌 Consideraciones
- Datos tratados bajo normativas de protección de datos
- Información usada únicamente para seguridad y atención médica
